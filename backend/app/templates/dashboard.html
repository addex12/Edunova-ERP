Root Route

GET /

Displays a welcome message and lists all available endpoints.

User Registration

POST /api/auth/register

Register a new user (student, teacher, parent, or admin).

User Login

POST /api/auth/login

Authenticate and receive a JWT token.

Dashboard

GET /api/dashboard

View user-specific dashboard data (requires authentication).

Admin Endpoints

GET /api/admin/users

List all users (admin only).

DELETE /api/admin/users/{id}

Delete a user (admin only).

Report Card Endpoints

POST /api/reports/templates

Create a new report card template (admin/teacher only).

POST /api/reports/generate/{student_id}

Generate a report card for a student (admin/teacher only).

Data Import/Export

POST /api/data/import/students

Import student data from a CSV file (admin only).
